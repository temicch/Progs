Команда SELECT

 Внутри <FORM> ... </FORM>, может присутствовать любое количество команд SELECT, в сочетании с другими HTML-элементами (включая INPUT и TEXTAREA) и текстом (кроме команды FORM). Команда SELECT представляется как список.

SELECT имеет закрывающую команду. Внутри оператора SELECT, разрешена только последовательность команд OPTION - за каждым из которых следует некоторое количество простого текста (без HTML разметки); например: 

 <SELECT NAME="my-menu">
 <OPTION VALUE="1" SELECTED> First option.
 <OPTION VALUE="2"> Second option.
 </SELECT>

Атрибуты команды SELECT:

NAME=идентификатор
символическое имя для этого SELECT элемента. Это поле должно присутствовать, т.к. оно используется при посылке запроса (аналогично команде INPUT).
SIZE=n
если SIZE равен 1 или если этот атрибут опущен, по умолчанию SELECT будет представлен как выпадающее меню. Если SIZE=2 или более, SELECT будет представлен как окно выбора; значение SIZE тогда будет определять, сколько элементов списка будут видны.
MULTIPLE
если присутствует (нет значения), задает, что SELECT должен позволять множественный выбор из списка. Наличие MULTIPLE принуждает SELECT быть представленным как список выбора, вне зависимости от значения SIZE.

Атрибуты команды OPTION:

VALUE="значение"
задается то значение, которое будет передано программе, обрабатывающей данную форму при выборе одного из элементов формы.
SELECTED
задает, что эта опция выбрана по умолчанию. Если SELECT позволяет множественный выбор (с помощью атрибута MULTIPLE), как SELECTED могут быть помечены несколько опций.

Динамическое изменение списка вариантов

Как уже говорилось выше, Команда SELECT представляется как список вариантов, который предоставляет пользователю два и более наименований на выбор. Наименования указываются в списке вариантов при его создании. Но с помощью функций языка JavaScript возможно и изменить содержимое списка вариантов.

Допустим, что нужно предоставить пользователю вариант с выбором либо машины, либо мотоцикла, но не двух предметов сразу. Можно сделать это путем создания двух кнопок-переключателей - Cars (Машины) и Motorcycles (Мотоциклы). При нажатии одной из таких кнопок вторая автоматически отключается. Другими словами, при нажатии кнопки Cars кнопка Motorcycles отключается, так как обе эти кнопки являются частью одной формы и имеют одно значение атрибута name.

Можно дать пользователю бо́льшую свободу и изменить наименование в списке вариантов, основываясь на выборе кнопки-переключателя пользователем. Иначе говоря, список вариантов отображает машины при нажатии кнопки Cars и мотоциклы при нажатии кнопки Motorcycles. Возможно динамически менять наименование в списке с помощью функций языка JavaScript при выборе другой кнопки-переключателя.

Вот пример того, каким образом это работает.

Создем форму и присваиваем ее атрибуту name значение, например, "Contact". Атрибуту name тега <select> присвоим значение "OptionList". Далее создаем список вариантов с, например, двумя моделями мотоциклов. За списком следуют две кнопки-переключателя, Cars и Motorcycles. Кнопка Motorcycles выбрана по умолчанию. Делается это следующим образом:
<input type="radio" name="vehicles" checked="true" value=1 onClick="ResetOptionList(this.value);">Motorcycles

Здесь атрибуту checked присвоено значение true - это и означает, что эта кнопка выбрана по умолчанию. Строчка
type="radio"
означает, что в качестве кнопки-переключателя мы используем окружности. Функция ResetOptionList(ElementValue) устанавливает список вариантов, соответствующий той кнопке, какую нажал пользователь. ElementValue - это параметр этой функции, определяющий номер нажатой кнопки. Здесь мы передаем этой функции параметр this.value, т.е., собственно, значение атрибута value тега <input>, т.е. номер кнопки.

Смена списка вариантов в функции ResetOptionList(ElementValue) осуществляется следующим образом:
function ResetOptionList(ElementValue) { with(document.forms.Contact) { if (ElementValue==1) { OptionList[0].text = 'Classic'; OptionList[0].value = 1; OptionList[1].text = 'Police Cruiser'; OptionList[1].value = 2; } ... } }
Ниже аналогично делается для кнопки Cars.
Здесь OptionList[...].text определяет название модели мотоцикла, а OptionList[...].value - номер варианта.
