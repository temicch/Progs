Работа с текстом

На Web-странице можно создать бесконечное число эффектов "переката", но каждый из них должен быть функционально оправданным. Например, rollover (при наведении мышкой появляется картинка), размещенные с умом, могут быть весьма полезны для посетителя.

Грамотное использование эффекта rollover Web-дизайнерами дает возможность посетителю увидеть дополнительную информацию о товаре, описанном в тексте. Таким образом, можно избежать создания гиперссылки на другую Web-страницу, содержащую дополнительную информацию, и сопутствующего избытка информации, который встречается на некоторых страницах.

Текстовый эффект rollover создается с помощью атрибута onmouseover тега <A>, который фактически является тегом anchor. Присваивание действия атрибуту onmouseover происходит таким же образом, что и в случае с тегом <IMG>.

Рассмотрим пример. Допустим, нужно сделать следующее: отображать обложку книги при наведении мыши на ее название. Для этого нужно сделать следующее: действие, указанное в атрибуте onmouseover, должно изменять значение атрибута src тега <IMG>. Следовательно, значение, присвоенное атрибуту onmouseover, должно точно определять тег <IMG>, который подвергается изменению. Для этого нужно дать тегу <IMG> определенное имя путем присваивания его атрибуту name тега <IMG>. Затем можно ссылаться на этот тег, указывая присвоенное имя, из действия, описанного в атрибуте onmouseover тега <A>. В следующем примере показано, как это возможно сделать.

Для начала мы даем название тегу <IMG>. Назовем его cover.
<img height="92" src="Java.gif" width="70" border="0" name="cover">

Затем мы используем присвоенное имя при изменении значения атрибута src тега <IMG> для смены изображения обложки. Заметьте, что мы используем полный путь, состоящий из названия документа, объекта внутри документа (тег <IMG> и атрибута объекта (src), который подвергается изменению. Нет смысла возвращать предыдущее значение при наступлении события onmouseover, так как изображение обложки всегда сопутствует той книге, на название которой был наведен указатель мыши.
<a onmouseover="document.cover.src='Java.gif'"> <b><u>Раскрытие тайн JavaScript</u></b> </a>
